# pioneer-ros
ROS packages to control Pioneer 3 mobile robots and their simulations in MobileSim and Gazebo

## 1. Environment
- Ubuntu 16.04, ROS Kinetic Kame 
- Ubuntu 18.04, ROS Melodic Morenia
